8x2 SRAM Design & Layout

2-byte SRAM design project with schematics and silicon layout implemented in Cadence Virtuoso.

Project Overview

This SRAM (static RAM) is a method of storing bits of data using transistors. This paper outlines the design of a 2-byte memory which uses 2 columns of 8 SRAM cells, giving a total of 16 bits of information. This includes both schematics and the silicon layouts.

Implementation

This project was implemented in Cadence Virtuoso using a 45 nm PDK.
