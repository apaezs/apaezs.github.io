Embedded Piano Synthesizer

Real-time audio synthesis using DAC, DMA, events, and USART on AVR

Project Overview

This project was the final portion of Lab 8 for EEL 4744C: Microcontroller Applications at the University of Florida. The goal was to design a real-time embedded piano synthesizer using the ATxmega128A1U microcontroller.

Musical notes are generated by streaming waveform lookup tables to the DAC using DMA triggered by timer-driven events, enabling continuous, glitch-free audio playback without CPU intervention. A connected computer keyboard acts as a piano interface via USART, allowing real-time note selection and waveform mode switching.

Hardware & Platform

  • MCU: Atmel ATxmega128A1U
  • Clock: 32 MHz system clock
  • DAC: Internal DAC (Port A)
  • Audio Output: IS31AP4991 amplifier + onboard speaker (OOTB Analog Backpack)
  • Input: PC keyboard via USART
  • Measurement Tools: Digilent Analog Discovery (WaveForms)

System Architecture

  • Waveform generation using precomputed lookup tables (LUTs)
  • Timer/Counter generates periodic events at precise sample rates
  • Event System triggers DAC conversions and DMA transfers
  • DMA Controller streams waveform samples directly from memory to the DAC
  • USART RX interrupts handle real-time keypress input
  • Minimal CPU usage during continuous audio playback

Audio & Synthesis Features

  • Supports 12 musical notes (C6 → C7) mapped to computer keyboard keys
  • Accurate note frequencies (≤ ±2% error)
  • Two waveform modes:
    • Sine wave
    • Triangle wave
  • Real-time waveform switching using keyboard command ('s')
  • Notes play only while the key is held, with no audible latency
  • Continuous, glitch-free audio output using DMA-driven playback
