Surrogate-Accelerated Genetic Optimization for Photonic Coupler Design

Neural-network-assisted evolutionary optimization of grating couplers using Lumerical FDTD

Project Overview

This project investigates a surrogate-assisted inverse design framework for optimizing photonic grating couplers. Traditional photonic optimization techniques are computationally expensive due to the complex electromagnetic interactions governing light–matter coupling. To address this challenge, we combine deep neural networks (DNNs) with a genetic algorithm (GA) to accelerate exploration of the design space :contentReference[oaicite:1]{index=1}.

A neural network is trained on data generated from a full-parameter sweep of Lumerical FDTD simulations. Once trained, the model serves as a fitness function within a genetic algorithm, enabling efficient evaluation of new designs without requiring full electromagnetic simulation at every iteration.

Publication: SPIE Digital Library

Design Parameters & Dataset

  • Total grating height (htotal): 0.18–0.26 μm
  • Etch depth: 0.06–0.16 μm
  • Duty cycle: 0.4–0.8
  • Pitch: 0.4–0.8 μm

A nested parameter sweep over these four variables generated 6,561 FDTD simulations (9⁴). Coupling efficiency at 1550 nm was extracted as the output metric and used to train the neural network :contentReference[oaicite:2]{index=2}.

Neural Network Surrogate Model

  • Feedforward neural network
  • 4 input nodes (design parameters)
  • 4 hidden layers, 500 ReLU neurons each
  • Single output: predicted coupling efficiency
  • R²: 0.9938
  • MAE: 0.002096

The trained surrogate accurately predicts coupling efficiency and dramatically reduces computational cost compared to full FDTD evaluation :contentReference[oaicite:3]{index=3}.

Genetic Algorithm Optimization

  • Population size: 100
  • Generations: 50
  • Mutation rate: 0.1
  • Crossover rate: 0.8
  • Elite ratio: 0.2
  • Seed ratio: 0.3

The neural network was used as the fitness evaluator for each candidate design. High-performing individuals were selected, crossed over, and mutated to guide the population toward optimal coupling configurations :contentReference[oaicite:4]{index=4}.

Results

While the genetic algorithm did not surpass the absolute best design found in the original dataset, it successfully generated new designs within 1% of the highest transmission using significantly less computational effort than traditional parameter sweeps or particle swarm optimization.

The best GA-predicted design achieved a coupling efficiency of 37.8000%, with a full FDTD-verified efficiency of 37.5733%, demonstrating strong surrogate generalization :contentReference[oaicite:5]{index=5}.
